# Quality Review System

A comprehensive system for managing quality reviews, checklists, and defect tracking.


##  Project Structure

- **`backend/`**: Node.js + Express + Prisma (MySQL) API.
- **`frontend/`**: Flutter (Dart) web application.

##  Main Technologies

- **Frontend:** Flutter
- **Backend:** Node.js, Express
- **Database:** MySQL, Prisma ORM
- **Authentication:** JWT, Cookies
- **Image Storage:** Local filesystem with metadata tracking
