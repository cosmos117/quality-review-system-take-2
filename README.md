# Quality Review System

A comprehensive system for managing quality reviews, checklists, and defect tracking.

## 🚀 Getting Started

If you are a new developer joining the project, please follow the **[Getting Started Guide](./GETTING_STARTED.md)** to set up your local environment.

## 📁 Project Structure

- **`backend/`**: Node.js + Express + Prisma (MySQL) API.
- **`frontend/`**: Flutter (Dart) web/mobile application.

## 🌐 Deployment & LAN Access

To make the app accessible to others on your internal office network, refer to the **[LAN Deployment Guide](./LAN_SETUP.md)**.

## 🛠️ Main Technologies

- **Frontend:** Flutter
- **Backend:** Node.js, Express
- **Database:** MySQL, Prisma ORM
- **Authentication:** JWT, Cookies
- **Image Storage:** Local filesystem with metadata tracking
